# Support

## How to Get Help

If you need help with this project:

- **Check the documentation**: Start by reading the [README](README.md) — it covers the project overview and goals.
- **Search existing issues**: Browse [open issues](https://github.com/my-mona-org/my-awesome-repo/issues) to see if your question has already been answered.
- **Open a new issue**: If you can't find an answer, open a new issue with as much context as possible.

## Reporting Bugs

To report a bug, please open an issue and include:

- A clear description of the problem
- Steps to reproduce the behaviour
- What you expected vs. what actually happened
- Any relevant environment details (OS, version, etc.)

See [SECURITY.md](SECURITY.md) for reporting security vulnerabilities privately rather than in a public issue.

## Contributing

We welcome contributions of all kinds! See [CONTRIBUTING.md](CONTRIBUTING.md) for guidelines on how to get involved.
