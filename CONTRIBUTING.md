# Contributing to my-awesome-repo

Thank you for your interest in contributing! 🎉 All contributions are welcome and appreciated.

## Getting Started

1. **Fork** this repository and **clone** your fork locally.
2. Create a new branch for your changes:
   ```bash
   git checkout -b my-feature-or-fix
   ```
3. Make your changes, commit them with a clear message, and push your branch.
4. Open a **Pull Request** against the `main` branch.

## How to Contribute

### Reporting Bugs

- Check existing [issues](../../issues) first to avoid duplicates.
- Open a new issue with a clear title and description, steps to reproduce, and expected vs. actual behaviour.

### Suggesting Features

- Open an issue describing the feature, its motivation, and any relevant use cases.
- Discussion first helps avoid wasted effort!

### Submitting Code

- Keep pull requests **small and focused** — one concern per PR.
- Write clear commit messages.
- Add tests if applicable.
- Ensure your changes don't break existing functionality.

## Code of Conduct

Please be respectful and constructive. We want this to be a welcoming space for everyone.

## Questions?

Feel free to open an issue if you have any questions. We're happy to help!
